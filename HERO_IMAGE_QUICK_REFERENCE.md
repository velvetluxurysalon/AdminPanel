# Hero Image Upload - Quick Reference Card

## Admin Panel Hero Section - Quick Start

### 📋 What's New

✅ Upload hero images for each slide  
✅ Add multiple images per slide  
✅ See image previews in admin panel  
✅ Drag to reorder slides  
✅ Works perfectly with website frontend

---

## 🎯 How to Add a Hero Slide

### Step 1: Click "Add Slide"

- Located in top-right of Hero Slides section

### Step 2: Fill Required Fields

| Field               | Example                             | Note     |
| ------------------- | ----------------------------------- | -------- |
| **Title**           | "Where Luxury Meets Beauty"         | Required |
| **Subtitle**        | "Experience world-class treatments" | Required |
| **Hero Image**      | PNG/JPG file                        | Required |
| **CTA Button Text** | "Schedule Now"                      | Optional |
| **CTA Button Link** | "/appointments"                     | Optional |

### Step 3: Upload Image

1. Click "Choose File" in Hero Image(s) section
2. Select JPG, PNG, or WebP from your device
3. Wait for upload confirmation (green success message)
4. Image preview appears in gallery

### Step 4: Save

- Click **"Add Slide"** button
- Confirmation message appears
- Slide now visible on website!

---

## ✏️ How to Edit a Slide

1. Click **"Edit"** button next to slide
2. Modify title, subtitle, or text
3. **Add more images**: Click file input and upload
4. **Remove images**: Click × on image thumbnail
5. Click **"Save Changes"**

---

## 🔄 How to Reorder Slides

1. Hover over slide in list
2. Click and drag the **grip handle** (⋮⋮ icon on left)
3. Drop in new position
4. Automatically saves!

---

## 🗑️ How to Delete a Slide

1. Find slide in list
2. Click **red "Trash"** icon on right
3. Confirm deletion
4. Slide removed (can't undo!)

---

## 📸 Image Guidelines

| Aspect         | Specification           |
| -------------- | ----------------------- |
| **Format**     | JPG, PNG, WebP          |
| **Size**       | Under 2MB               |
| **Dimensions** | 1920×1080 (16:9)        |
| **Min Width**  | 1440px                  |
| **Quality**    | High quality/compressed |

### ✅ Good Image

- Professional photo
- Clear and sharp
- Properly lit
- Compressed to 1-2MB

### ❌ Avoid

- Blurry images
- Very large files (>5MB)
- Wrong aspect ratio
- Low resolution

---

## 🆘 Troubleshooting

### "Image failed to upload"

- Check file size (< 2MB)
- Try different format (JPG instead of PNG)
- Check internet connection
- Retry upload

### "Slide failed to save"

- Ensure all required fields filled
- Check at least one image uploaded
- Try refreshing page
- Contact support if persists

### "Images not showing on website"

- Hard refresh website (Ctrl+Shift+R)
- Wait 2-3 minutes for cache update
- Check different browser
- Contact support

### "Multiple images not showing correctly"

- This is normal! Images overlap by design
- Last uploaded image appears on top
- Transparent/opaque images layer nicely
- Check website for visual effect

## ❓ FAQ

**Q: Can I upload multiple images?**  
A: Yes! Each image is a layer - they'll stack nicely.

**Q: What if I delete a slide?**  
A: It's removed from website. Can't be recovered!

**Q: Can I edit image after upload?**  
A: No, but you can delete & re-upload new version.

**Q: How many slides can I have?**  
A: As many as you want! 3-10 is recommended.

**Q: Do images scale on mobile?**  
A: Yes! Automatically responsive.

**Q: Why are images so large?**  
A: Compress before upload using:

- Online: TinyPNG.com or similar
- Windows: Paint → Save as → Quality 80%
- Mac: Preview → Tools → Adjust Size
